## ğŸ¯ Getting to Specific Features

**Want to add a new page?**
1. Create `src/pages/NewPage.jsx`
2. Create `src/pages/NewPage.css`
3. Add route in `src/App.js`

**Want to add a new component?**
1. Create `src/components/NewComponent.jsx`
2. Create `src/components/NewComponent.css`
3. Import and use in pages

**Want to change colors?**
1. Edit CSS variables in `src/styles/global.css`

**Want to add API endpoint?**
1. Add to `src/api/client.js`
2. Use in component with `import { apiFunction } from '../api/client'`

**Want to add helper function?**
1. Add to `src/utils/helpers.js`
2. Import where needed
